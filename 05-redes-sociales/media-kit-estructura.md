# Estructura de Media Kit Agora / Elenxos

Este documento define los activos visuales mínimos que debe contener el Media Kit oficial para que la marca se vea establecida y profesional ante aliados, prensa y usuarios.

## 1. Identidad Visual (Brand Assets)
- [ ] **Logos:** Versión principal, versión horizontal, versión vertical e isotipo (en formatos SVG, PNG transparente y JPG).
- [ ] **Variantes de Color:** Logo en versión luz (blanco) y versión oscuridad (negro/azul profundo).
- [ ] **Paleta de Colores:** Código Hex, RGB y CMYK de los colores oficiales (Azul Profundo, Verde Menta, Gris Oxford, Blanco Puro).
- [ ] **Tipografías:** Enlace de descarga o archivos de las fuentes oficiales (Lora/Playfair para Elenxos y Montserrat/Inter para Agora).

## 2. Capturas de Pantalla (Product Shots)
- [ ] **Hero Shot:** Captura limpia del dashboard de Agora en alta resolución.
- [ ] **Módulo de Editor Semántico:** Zoom en el editor con un texto en Markdown y lógica ST visible.
- [ ] **Módulo de Terminal:** Captura de la terminal multi-sesión con una ejecución real.
- [ ] **Módulo de Colaboración:** Vista de tableros Kanban o lista de miembros del equipo.
- [ ] **Dispositivos:** Mockups de Agora funcionando en un MacBook, un iPad y un iPhone (para denotar PWA & Offline).

## 3. Gráficos de Redes Sociales (Social Media Templates)
- [ ] **Banners:** LinkedIn, X y YouTube (versión Elenxos y versión Agora).
- [ ] **Plantilla para Carruseles:** Diseño base para LinkedIn/Instagram con la flecha de flujo `Escribe → Formaliza → Verifica → Colabora`.
- [ ] **Plantilla para "Caso Verificado":** Diseño para testimonios con espacio para foto, cargo e institución.
- [ ] **Fondo de Videollamada:** Un fondo profesional con el logo de Agora/Elenxos para demos y reuniones.

## 4. Fotos de Equipo (Human Assets)
- [ ] **Retrato de Fundador:** Foto profesional en alta resolución.
- [ ] **Workspace:** Foto del equipo trabajando o de un escritorio con Agora abierto (para "humanizar" la marca).

## 5. Especificaciones Técnicas de Entrega
- **Imágenes:** Formato PNG para capturas de pantalla, JPG para fotos de equipo y SVG para logos.
- **Videos/Reels:** Formato MP4, 1080x1920 (9:16), máximo 60 segundos.
- **Compresión:** Todas las imágenes para web deben estar optimizadas (usar TinyPNG o similar antes de subir al repo).
