# Identidad y Marca

Este directorio concentra la capa canonica de identidad de Elenxos y Agora.

La regla es simple:

- aqui se define el sistema visual y verbal
- la ejecucion diaria de publicaciones vive en `../03-operacion-redes/`
- si una decision afecta piezas publicas, debe quedar alineada con `../assets/README.md`
