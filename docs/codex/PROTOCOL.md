# symploque Codex Protocol

## Objective

Drive work to completion with minimal supervision while keeping decisions, validation, and blockers visible.

## Default Behavior

- Inspect before editing.
- Prefer direct implementation over advisory-only replies.
- Validate meaningful changes before closing the loop.
- Update `WORKLOG.md` after substantial milestones.
- Update `NEXT_ACTIONS.md` whenever work remains open.

## Pause Conditions

Pause only for:

- Destructive changes not already requested
- External state changes such as deploy, publish, or push
- Ambiguous product or business decisions
- Secrets, credentials, or production data access
